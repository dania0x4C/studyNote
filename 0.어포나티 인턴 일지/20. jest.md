1. only, skip
	- 테스트 코드를 디버깅 할때 사용
	```
		it.only 이렇게 작성하면 이 테스트만 진행함
		it.skip 이건 이 테스트만 빼고 진행함
	```

3. mocking
	- 테스트 하고자 하는 코드가 의존하는 function, class에 대해서 모조품으로 일단 돌아가게 만드는 것이다.
	- 즉, 디비가 잘 돌아간다는 전제로 데이터베이스 저장 메소드를 mock 시키는 것이다.
